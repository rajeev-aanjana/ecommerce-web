# 🛍️ Modern Ecommerce Frontend Application

A fully responsive modern Ecommerce Frontend Application built using React.js, Redux Toolkit, Tailwind CSS, Custom Hooks, React Router DOM, and DummyJSON API.

This project focuses entirely on frontend development and demonstrates modern React architecture, reusable components, state management, filtering, routing, pagination, search optimization, and responsive UI/UX.

Implemented debounced search to avoid unnecessary API calls.

### Logic Used:

* useEffect
* setTimeout
* clearTimeout

* Performance

* User experience
